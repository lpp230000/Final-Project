# Final-Project
Final Project for Program in Digital Arts (2025) includes proposal & project
## Project Description
This project is a simple platform game where the orb, the player,can jump onto randomized platforms located in front of a waterfall. In addition there is use of the digital rain code to provide the atmosphere of rain.
